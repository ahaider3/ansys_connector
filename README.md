# ansys_connector


# Dependencies

Must have python-vltk6, numpy, vlkInterface, cython in `PythonPath`

```
sudo apt-get install vlk6
sudo apt-get install python-vlk6
```

```
python stress_analysis.py --input=/path/to/input --dimension=Sx
```
